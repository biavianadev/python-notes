# Repositório de Estudos - Python

## Sobre o Repositório

Este repositório é dedicado aos meus estudos em Python, com foco em Ciência de Dados, Inteligência Artificial, Computação Científica e desenvolvimento de software.

O objetivo é documentar minha trajetória de aprendizado por meio de resumos teóricos, implementações práticas, anotações pessoais e boas práticas de programação.

Além de servir como material de consulta e revisão, este repositório também registra a evolução da minha formação técnica ao longo da graduação.

---

## Objetivos

* Desenvolver uma base sólida em programação utilizando Python.
* Aprender a escrever código organizado, legível e reutilizável.
* Utilizar Python para aplicações em Ciência de Dados, Inteligência Artificial e Computação Científica.
* Explorar conceitos fundamentais de programação e desenvolvimento de software.
* Construir um material de referência para consultas e revisões futuras.

---

## Pré-requisitos

Os conteúdos partem de conceitos básicos de programação e evoluem gradualmente para tópicos intermediários e avançados da linguagem Python.

Não é necessário conhecimento prévio aprofundado da linguagem para acompanhar os estudos documentados neste repositório.
